## **开发调试接口连接**
Icrest3可以提供3种方式进行开发调试，用户可根据实际需求进行硬件连接
1. DEBUG UART有线连接
2. OTG 有线连接（仅M300RTK）
3. WIFI连接
4. 以太网有线连接
